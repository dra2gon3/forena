Forena Reports

Enterprise Reporting module that provides a mechanism for writing reports 
in a modified html syntax.  This module really leverages XML, and for this 
reason required PHP 5.2 in order to run correctly. 

Installation
------------

Copy the forena project into your module directory and then enable on the admin 
modules page.  Canned reports are enabled that provide some basic functionality.
